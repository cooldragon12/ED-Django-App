## Getting Started
### Dependencies
* Django 3.8 or latest
* Pillow
* Django Rest Framework (For Serializing)
* MySQLClient
* MySQL Database
* Django-Filters
* Selenium
* uWSGI

* Step-by-step Setup


```
py manage.py migrate
```
